# webapp-gradle-gae

Simple GAE template for web application  - java8, gradle, kotlin, spark


Deploy the app

```sh
gradle clean appengineUpdate
```

demo : 
```
https://spark-demo-dot-myproject.appspot.com/
```

---Development----

Run the DEV server 
```sh
gradle clean appengineRun
```

Stop the DEV server 
```
gradle clean appengineStop
```
