# webapp-gradle-gae

Starter web application template for GAE - java8, gradle, kotlin


Deploy the app

```sh
gradle clean appengineUpdate
```

demo : 
```
https://demo-dot-myproject.appspot.com/
```

---Development----

Run the DEV server 
```sh
gradle clean appengineRun
```

Stop the DEV server 
```
gradle clean appengineStop
```
